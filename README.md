# 🛒 Ronn's TechStore

A fully responsive e-commerce web app for tech products built with pure HTML, Tailwind CSS, and Vanilla JavaScript.

## 🚀 Live Demo
[View Live Site](https://rania-essam.github.io/EcommerceTechStore/)

## ✨ Features
- 🔍 Browse products by category (Phones, Laptops, SmartWatches, Tablets, Accessories)
- 🛒 Add to cart with quantity control and local storage persistence
- 🔐 Register & Login system (localStorage based)
- 📦 Real product data from [DummyJSON API](https://dummyjson.com)
- ⌚ Custom smartwatch data
- 🔔 Toast notifications on cart actions
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Dark mode UI

## 🛠️ Built With
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- DummyJSON API
- Unsplash (product images)

## 📁 Project Structure
├── index.html       # Main HTML file
├── script.js        # All JavaScript logic
└── Images/          # Local assets
